# vemreyilmaz.com

Personal portfolio website built with Next.js 16, React 19, and Tailwind CSS v4.

## Tech Stack

- **Framework:** Next.js 16 (Static Export)
- **UI:** React 19, Tailwind CSS v4, Framer Motion
- **Blog:** MDX with rehype-pretty-code syntax highlighting
- **Components:** shadcn/ui, Lucide Icons
- **Deployment:** Docker + CI/CD

## Features

- Project showcase with detail pages
- MDX-powered blog with syntax highlighting
- Dark/light theme toggle
- Contact form (Formspree)
- Fully responsive design
- SEO optimized with Open Graph support

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```

Static files are generated in the `out/` directory.
